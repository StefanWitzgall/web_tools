Usage Manual

SETUP Define constants and animation behavior. Definitions here replace strings globally in functions.

LIVEUPDATE What you type is read instanly, e.g., names and values.

ANIMATION Use Start/Stop button or the Spacebar.

FUNCTIONS Create \bfun ... \efun blocks. You can reference previously defined functions by name.

VARIABLES x and t are variable and clock parameter for the functions.

MARKERS markers: yesMake the function to be plotted with a marker. Use at: x-value or domain: [x1 x2] to note where makers are set.

COMMENTS % starts a comment for the rest of the line.

PARSING Incomplete or unknown tokens are gracefully ignored (fall back to defaults). Key: Value pairs are separated by newline or semicolon ; They can be escaped with the backslash as in \;. Keywords ignore case, e.g., axisXmin = axisXmin, function names are casesensitive, i.e., f ≠ F

JAVA SCRIPT may be included, e.g., term: (() => { let m = 1\; if ( t>2 ){m = 3}\; return f' * m })() + (f / 2);.
